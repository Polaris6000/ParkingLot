[종합 프로젝트] 웹 기반 스마트 주차 관리 시스템 구축

1. 프로젝트 개요

* 주제: JSP, Servlet, JDBC를 활용한 주차 관제 시스템
* 목표:
- MVC 패턴 기반의 웹 애플리케이션 구조 설계
- CRUD(주차관리, 회원 관리)와 비즈니스 로직(요금 계산)의 통합 구현
- 데이터베이스 집계 함수(`SUM`, `COUNT`, `GROUP BY`)를 활용한 통계 페이지 구현
- 사용자 편의 기능(영수증, 시각적 UI) 구현
- Git/GitHub을 활용한 형상 관리 및 소스 코드 공유 능력 배양
- 팀원 간 역할 분담 및 코드 병합(Merge) 과정에서의 충돌 해결 능력 습득


2. 개발 환경

* 언어 & 기술: Java, JSP, Servlet
* DBMS: MariaDB (또는 MySQL)
* Front-end: HTML, CSS, JavaScript (Modal, Chart 라이브러리 허용)

3. 운영 정책 (비즈니스 로직)
3.1 기본 시설 정보

* 주차장 명: 스마트주차 반월당점
* 총 주차 면수: 20면 (구역 번호: A-1 ~ A-20)
* 운영 시간: 24시간 연중무휴

3.2 요금 및 할인 정책 (DB 관리 필수)

1. 회차(무료) 시간: 입차 후 10분 이내 출차 시 요금 0원
2. 기본 요금: 최초 1시간 2,000원 (10분 초과 ~ 1시간 이내)
3. 추가 요금: 이후 30분당 1,000원
4. 일일 최대 요금: 15,000원 (입차 시점부터 24시간 기준)
5. 할인 혜택:
- 월정액 회원: 무료 (0원)
- 장애인: 50% 할인
- 경차: 30% 할인
- 중복 할인 불가 (높은 할인율 하나만 적용, 월정액 우선)


4. 기능 요구사항 상세

4.1 주차 현황 (메인 대시보드)

* 구역 시각화: 20개의 주차 구역(A-1 ~ A-20)을 박스 형태 UI로 표현.
* 상태 표시:
* 빈 자리: '사용 가능' 표시 (클릭 시 입차 화면 이동)
* 사용 중: 차량 번호 및 입차 시간 표시 (클릭 시 출차/상세 이동)

* 검색: 차량 번호 뒷 4자리로 현재 주차된 위치 검색.

4.2 차량 입차 관리

* 정보 입력: 차량 번호, 차종(일반/경차/장애인), 연락처.
* 구역 지정: 입차 시 원하는 구역(A-1 ~ A-20) 선택 (이미 점유된 구역 선택 불가).
* 회원 자동 인식: 차량 번호 입력 시 `MEMBERS` 테이블을 조회, 월정액 회원이면 화면에 "회원(무료 적용)" 표시.

4.3 차량 출차 및 정산 (영수증 기능)

* 요금 계산: 회차 시간(10분) 처리 및 할인 정책 적용 로직 수행.
* 영수증 팝업(Modal): 출차 완료 시 자바스크립트 팝업으로 [입/출차 시간, 총 주차 시간, 할인 내역, 최종 금액] 출력.

4.4 월정액 회원 관리 (CRUD 필수)

* 회원 목록 조회: 등록된 전체 회원 리스트 표시 (페이징 처리는 필수).
* 회원 등록:
- 필수 정보: 차량번호, 이름, 연락처, 시작일, 만료일.
- 유효성 검사: 이미 등록된 차량 번호는 중복 등록 불가.

* 회원 수정: 연락처 변경 및 기간 연장(만료일 갱신) 기능.
* 회원 삭제: 회원 정보 삭제 또는 비활성화 처리.

4.5 관리자 & 통계 대시보드

* 설정 관리: 기본 요금, 추가 요금, 할인율 수정 기능.
* 매출 통계 (Dashboard):
1. 일별 매출: `GROUP BY`를 사용하여 날짜별 총 결제 금액 및 입차 대수 집계.
2. 월별 매출
3. 차종별 비율: 이용 차량 유형 비율 (일반/경차/장애인) 그래프 또는 표.
